# Finance Benchmark Preset

Use this config template as a starting point for finance-domain benchmark runs.

Example:
```bash
python eval/benchmark_runner.py \
  --config eval/benchmarks/finance/benchmark_config.finance.example.json \
  --compare-with eval/benchmark_config.previous.example.json \
  --output benchmark_finance.md \
  --output-json benchmark_finance.json
```
